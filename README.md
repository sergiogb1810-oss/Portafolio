📄 Descripción

Desarrollé una aplicación web utilizando Django que permite gestionar proyectos y tareas, incluyendo funcionalidades como creación, edición y visualización de datos.

⚠️ Desafío principal

El principal desafío fue implementar correctamente la relación entre proyectos y tareas, además de manejar las rutas, vistas y plantillas sin errores.

💡 Solución propuesta

Se desarrolló una aplicación basada en el patrón MVT de Django, utilizando modelos relacionados (ForeignKey), vistas funcionales y templates HTML para mostrar la información correctamente.
